配置gitalk

配置分两个步骤

  1. 在Github上配置OAuth APP

    • 登陆你的github，点击右上角你的头像，进入setting

    • 左边：Developer settings

    • 左边：OAuth Apps

    • 右上按钮：New OAuth App

      然后你就来到了 这个页面

    Application name:随便填一个应用名称
