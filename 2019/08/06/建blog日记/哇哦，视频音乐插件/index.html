哇哦，视频音乐插件

很cooooool
可以插入音频诶
可以大大方方把我的拥有酷炫effects的动画放上去啦

参考感谢：https://www.jianshu.com/p/26a7fc7cc185

视频插件

安装（假设您已经安装了cnpm）

  1. 切到博客根目录
  2. Git Bash Here
  3. 输入cnpm install hexo-tag-dplayer --save

使用

例子：
{% dplayer “url=123.mp4” “pic=http://home.ustc.edu.cn/~mmmwhy/GEM.jpg" “loop=yes” “theme=#FADFA3” “autoplay=false” “volume=0.8”%}
插入到文章任意处
@parama

  • url 视频位置，可以是本地视频，可以是互联网视频
  • pic 视频封面位置，可以是本地图片，可以是互联网图片
  • loop 是否默认开启无脑循环
  • theme 进度条颜色
  • autoplay 是否自动播放

更多：https://github.com/MoePlayer/hexo-tag-dplayer

音乐插件

安装（假设您已经安装了cnpm）

  1. 切到博客根目录
  2. Git Bash Here
  3. 输入cnpm install hexo-tag-aplayer --save
    例子：
    {% aplayer “她的睫毛” “周杰伦” “http://home.ustc.edu.cn/~mmmwhy/%d6%dc%bd%dc%c2%d7%20-%20%cb%fd%b5%c4%bd%de%c3%ab.mp3" “http://home.ustc.edu.cn/~mmmwhy/jay.jpg" “autoplay=false” %}
